📚 Aprendi:
• As condicionais são usadas para fazer decisões no programa, sendo elas:
└ (if): Significa (Se... então);
└ (else): Significa (Senão... então);
└ (else if, elif): Significa (Senão, se... então);
└ Estrutura principal:
/*
if (condição)
{
    (ação);
}
*/
└ Estrutura simples: utiliza apenas o if;
└ Estrutura composta: utiliza if e else;
└ Estrutura encadeada: utiliza if, else if e else.

• Switch é uma estrutura de decisão usada para comparar uma única variável com diferentes valores possíveis, podendo substituir vários if e else if quando as comparações são específicas;
└ Estrutura principal:
/*
switch (variável)
{
    case valor:
        (ação);
        break;
}
*/

• Char é um tipo de dado que armazena apenas um único caractere, normalmente utilizando aspas simples (' '), diferente da string, que armazena textos utilizando aspas duplas (" ").

• Métodos (void) são usados para organizar ações específicas do programa, podendo ser chamados sempre que necessário;
└ Estrutura principal:
/*
void NomeDoMetodo()
{
    (ação);
}
*/
└ Para executar um método:
/*
NomeDoMetodo();
*/

• O método Start() é executado automaticamente uma única vez quando o jogo inicia;
└ Estrutura principal:
/*
void Start()
{
    (ação);
}
*/

• O método Update() é executado continuamente enquanto o jogo estiver rodando;
└ Estrutura principal:
/*
void Update()
{
    (ação);
}
*/

• O operador lógico (!) significa (NÃO), invertendo o valor de uma variável booleana;
└ Exemplo:
/*
bool portaAberta = true;

if (!portaAberta)
{
    Debug.Log("A porta está fechada.");
}
*/

• Métodos void não retornam valores, por isso não podem ser utilizados diretamente dentro de um if;
└ Estrutura incorreta:
/*
if (PegarAChaveDaGaveta())
{
    (ação);
}
*/
└ Estrutura correta:
/*
PegarAChaveDaGaveta();

if (temChave)
{
    (ação);
}
*/

• Aprendi a organizar um projeto dividindo-o em pequenos sistemas com métodos:
└ Cabine;
└ Chave;
└ Portas;
└ Cortinas;
└ Computador;
└ Câmeras.
