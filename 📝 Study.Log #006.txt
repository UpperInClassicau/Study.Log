📚 Aprendi:
• O (for) é uma estrutura de repetição utilizada quando sabemos (ou conseguimos definir) quantas vezes uma ação deve ser executada. Ele reúne em uma única linha a criação da variável de controle, a condição para continuar a repetição e a atualização dessa variável, tendo a estrutura principal:
/*
for (variável ou criação de uma variável; condição; atualização da variável)
{
(ação que será executada a quantidade de vezes definida pela condição)
}
*/

• O (while) é uma estrutura de repetição utilizada quando não sabemos exatamente quantas vezes uma ação será executada. Enquanto a condição for verdadeira, o bloco de código continuará sendo executado. A condição é verificada antes de cada repetição, tendo a estrutura principal:
/*
variável

while (condição)
{
(ação que será executada enquanto a condição for verdadeira)

atualização da variável;

}
*/

• O (do...while) é uma estrutura de repetição semelhante ao (while), porém a ação é executada primeiro e somente depois a condição é verificada. Por isso, o bloco de código será executado pelo menos uma vez, tendo a estrutura principal:
/*
variável

do
{
(ação que será executada pelo menos uma vez)

atualização da variável;

}
while (condição);
*/

• O (break) é utilizado para interromper imediatamente uma estrutura de repetição (for, while ou do...while);

• O (continue) é utilizado para pular apenas a repetição atual e continuar normalmente para a próxima repetição da estrutura de repetição.
